# Streamlit Tutorial

## What is Streamlit?
Streamlit is an open-source Python library that turns data scripts into shareable web apps in minutes. All in pure Python. No front-end experience required. It is widely used for creating dashboards, machine learning tools, and data visualizations quickly and easily.

## key Features
- **Embrace Scripting**: Build apps with just a few lines of code.
- **Interactions**: Add widgets like sliders, buttons, and text inputs with no callbacks.
- **Deploy Instantly**: Share your apps with the world easily.

## Installation
To get started with Streamlit, you need to install it using pip. Open your terminal or command prompt and run:

```bash
pip install streamlit
```

## How to Run
Once installed, you can run the Streamlit applications included in this repository.

To run the main application:
```bash
streamlit run app.py
```

To run the additional tutorial file:
```bash
streamlit run app1.py
```

After running the command, a new tab will automatically open in your default web browser showing the app.
